# mingkh
Parser mingkh.ru with user-friendly interface.<br>
This is a simple site parser https://mingkh.ru on Golang.

## installation
```golang
git clone https://github.com/RB-PRO/mingkh.git
```

## Build
```golang
go mod tidy
go build
```

### Run for Linux
```golang
./minzhhk
```

### Run for Windows
```golang
minzhhk.exe
```
